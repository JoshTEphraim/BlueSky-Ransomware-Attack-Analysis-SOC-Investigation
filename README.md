# BlueSky-Ransomware-Attack-Analysis-SOC-Investigation


## Overview
This project documents the investigation of the **BlueSky ransomware attack scenario** from CyberDefenders.

The objective was to analyze attacker behavior, identify indicators of compromise, and reconstruct the attack timeline using forensic artifacts and network analysis.

## Skills Demonstrated

- Network Forensics
- Malware Analysis
- Log Analysis
- Threat Hunting
- MITRE ATT&CK Mapping
- Incident Response Investigation

## Tools Used

- Wireshark
- CyberChef
- VirusTotal
- Windows Event Logs
- PowerShell Analysis

## Attack Summary

The attacker conducted reconnaissance, gained access to the target environment, executed malicious PowerShell commands, disabled security controls, and deployed ransomware to encrypt victim files.
